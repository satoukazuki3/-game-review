私はゲームを買う時パッケージに書いている情報を見たり、
気になるゲームを実況しているのを見ていると購買意欲がそそられて即購入することが
多く、ユーザーがゲームの思い出を書いてレビューしたり面白かったゲームの感想を書いて
レビューするといった個人の感情が大きいレビューサイトがあれば面白いのではと思い
このテーマにしました。
